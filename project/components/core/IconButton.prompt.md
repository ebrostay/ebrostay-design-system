One-line: Icon-only square/round button; the `glass` variant overlays on photos (favorite heart, gallery arrows).

```jsx
<IconButton label="Save" variant="glass" round><i data-lucide="heart" /></IconButton>
<IconButton label="Next" variant="secondary"><i data-lucide="chevron-right" /></IconButton>
```

Variants: `solid` · `secondary` · `ghost` · `glass`. Sizes `sm/md/lg`. `round` for circular.
